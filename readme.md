## How to install?

Just clone the repo, write a script in ScriptedBash and then use the script to translate it.

Example:

``` bash
git clone https://github.com/ririinsta/scriptedbash.git
cd scriptedbash
python3 scriptedbash.py examples/println.sb
chmod +x println.sh
./println.sh
```
